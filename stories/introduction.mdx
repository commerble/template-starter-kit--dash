# Welcome to Commerble Dash Storybook

Commerble Dash用のStorybookです。

ナビゲーション構成は以下の通りです。

*   Base
    *   Color Palette テーマカラーの一覧です。
    *   Layout ページレイアウトのサンプル実装の一覧です。
    *   Text テキストスタイルのサンプル実装の一覧です。
*   Components
    *   各コンポーネントの使用方法とサンプル実装があります。コントロールを使用して状態を変更しコピーして使用できます。
*   Pages
    *   テンプレートのモックHTMLを管理します。

このStorybookではCSSコンポーネントのみを開発・管理します。よって、JavasScriptはコンポーネントに含みません。
JavaScriptの実装はテンプレート実装者が責任を持ちます。
